# Design Studio Keyboard Shortcuts

## Tool Modes
| Key | Action |
|-----|--------|
| V | Select mode |
| R | Rectangle mode |
| L | Line mode |
| M | Move mode |
| T | Trim mode |
| D | Quick Dimension |
| H | Horizontal Dimension |
| J | Vertical Dimension |
| G | Toggle grid on/off |

## Selection & Editing
| Key | Action |
|-----|--------|
| Escape | Cancel current action, clear selection |
| Delete / Backspace | Delete selected shape(s) |
| Arrow keys | Nudge selected 1px |
| Shift + Arrow keys | Nudge selected 10px |

## Zoom
| Key | Action |
|-----|--------|
| + | Zoom in |
| - | Zoom out |
| 0 | Reset zoom to 1:1 |

## Ctrl / Cmd Combos
| Key | Action |
|-----|--------|
| Ctrl+Z | Undo |
| Ctrl+Shift+Z / Ctrl+Y | Redo |
| Ctrl+C | Copy selected |
| Ctrl+V | Paste |
| Ctrl+A | Select all |
| Ctrl+S | Save file |
| Ctrl+D | Duplicate selected (offset 20px) |

> Shortcuts are disabled when typing in input fields.
