# 2D Quotation - Keyboard Shortcuts Guide

## Quick Reference

Use these keyboard shortcuts to work faster in the 2D Quotation page.

---

## Export & Save

| Shortcut | Action | Description |
|:---------|:-------|:------------|
| `Ctrl+E` | **Export PDF** | Quickly export your quotation to PDF |
| `Ctrl+S` | **Save/Export** | Alternative shortcut to export PDF |

> **Tip:** Press `Ctrl+E` at any time to instantly export your work.

---

## Drawing & Selection

| Shortcut | Action | Description |
|:---------|:-------|:------------|
| `Ctrl+D` | **Toggle Draw Mode** | Enable or disable drawing mode |
| `Delete` | **Delete Unit** | Remove the currently selected unit |
| `Escape` | **Clear Selection** | Deselect units and exit draw mode |

> **Tip:** Use `Escape` to quickly cancel any operation or clear your selection.

---

## Navigation

| Shortcut | Action | Description |
|:---------|:-------|:------------|
| `Tab` | **Navigate Forward** | Move focus to the next interactive element |
| `Shift+Tab` | **Navigate Backward** | Move focus to the previous element |
| `Enter` | **Activate** | Click the focused button or link |

---

## Best Practices

### For Power Users

1. **Start Drawing:** Press `Ctrl+D` to enter draw mode
2. **Draw Units:** Click and drag on the canvas
3. **Adjust:** Select and modify units as needed
4. **Export:** Press `Ctrl+E` when done

### For Accessibility

- All keyboard shortcuts work without a mouse
- Screen readers announce all actions
- Focus indicators show your current position
- Skip links available at the top of the page

## Tips & Tricks

**Quick Export:** Press `Ctrl+E` to instantly generate and download a PDF of your current quotation.

**Rapid Prototyping:** Use `Ctrl+D` to quickly toggle draw mode on and off while sketching multiple units.

**Clean Slate:** Press `Escape` to clear everything and start fresh.

**Keyboard-Only Workflow:** You can complete an entire quotation without touching your mouse using Tab, Enter, and the shortcuts above.

---

## Troubleshooting

**Shortcut not working?**

- Make sure you're not in an input field (click the canvas first)
- Check that you're using the correct modifier key (Ctrl on Windows/Linux, Cmd on Mac)
- Ensure the page has focus

**Need to see all shortcuts?**

- This guide is always available in the documentation
- Screen reader users: Navigate to the page header for a list of available shortcuts
